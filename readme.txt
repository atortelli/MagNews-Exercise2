Dati di ingresso:
port --> porta del server smtp a cui collegarsi
smtp Host --> indirizzo del server smtp
sender email -->indirizzo email del mittente
sender password -->password dell'account mail del mittente
receiver email --> indirizzo email del destinatario
mail subject --> oggetto dell'email
mail size --> grandezza dell'email
number of email to send -->numero di email da mandare
Multiple connection/Unique connection --> con questa opzione si sceglie se inviare le mail con un unica connessione o con connessioni multiple (una per ogni mail)

Dopo aver inserito tutti i dati cliccare su "Send email(s)" per effettuare il benchmark. I risultati verranno mostrati sulla tabella di output. Ogni volta che si clicca su "Send mail(s)" viene aggiunta una riga alla tabella con i risultati del benchmark.
